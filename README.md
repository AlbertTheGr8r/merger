# Merger

Image manipulation tool to merge **tiled image** into one image.

Inspired by: [pnytko/merger](https://github.com/pnytko/merger)

## Requirements

```bash
python3
numpy
rich
pillow
```

## Using

Merging image.

```bash
python merger.py --path {path_to_folder_with_tiles} --out {output_path}
```
